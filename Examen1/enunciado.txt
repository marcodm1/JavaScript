
1. (6 puntos) Programar el contenido del fichero “index.js” y “leer.php” teniendo en cuenta los apartados:

    • Cada vez que se muestre la página se guardarán mediante Web Storage los datos necesarios para que la siguiente 
    vez se pueda mostrar cuánto tiempo ha transcurrido desde la visita anterior en días, horas y minutos, 
    así como la posición desde la que se conectó. Si no se puede obtener la localización en 3 seg. 
    se dejará la localización guardada anteriormente.

    • Cada vez que se muestre la página se mostrará el tiempo transcurrido desde la última vez y la posición desde la 
    que se realizó la conexión. Si es la primera vez que se muestra la página solamente aparecerá el mensaje oportuno.

    • En el área de datos, cada vez que se modifique el número de registro o que se pulse el botón “Leer registro” se 
    comprobará que el número está comprendido entre 0 y 20 y si cumple este requisito se realizará una llamada al fichero
     “leer.php” que devolverá el registro correspondiente. En caso de que el número de registro introducido no esté entre
      0 y 20 o “leer.php” no lo encuentre se mostrará un mensaje de error. Si lo encuentra, enviará los datos del registro
       (únicamente) para que desde JavaScript se muestre la tabla con los datos.

    • Hay que rellenar el código php necesario en el fichero “leer.php” para enviar los datos correspondientes 
    al número de registro introducido.
